# Smart Contract Audit Prep

This is not a professional audit.

Use this folder to prepare for an external audit.

## Pre-audit requirements

- [ ] Freeze contract scope.
- [ ] Write complete NatSpec.
- [ ] Complete unit tests.
- [ ] Complete fuzz tests.
- [ ] Run Slither.
- [ ] Run dependency audit.
- [ ] Document deployment/admin roles.
- [ ] Document pause/emergency controls.
- [ ] Document treasury withdrawal policy.
- [ ] Document upgradeability status.
- [ ] Document all privileged functions.
- [ ] Document known limitations.

## Required external review

Before deployment with real value:

- Independent smart contract audit
- Legal review of token design
- Treasury operations review
- Incident response plan
- Key ceremony / multisig setup
