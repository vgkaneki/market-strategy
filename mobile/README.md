# Mobile Strategy

Recommended mobile stack:

- React Native / Expo
- Read-only market overview first
- Watchlists
- Alerts
- Push notifications
- Wallet connection later
- Trading disabled until security/compliance approval

Mobile should not render the full Bookmap-style heatmap at first. Start with:

1. Watchlist
2. Symbol detail
3. Simplified chart
4. Active scanner levels
5. Alert management
6. Push notifications
